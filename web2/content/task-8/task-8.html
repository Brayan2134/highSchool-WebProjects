Web-Val.

What is website validation?

Website validation is the evaluation of a static site, to make sure that if conforms to various organizations standards and makes sure that the webpage is built to a spec, which can be maintained and scalable.

Why Validate?

Like stated above, it's to make sure that the website is able to be maintaned and scaled if desired. It's very strict on code readability, and neatness so any developer in the future can quickly make and push changes to the website. It's also used in Google's SEO, so having little to no errors is critical. Website validation also helps teach developers good habits when dealing with any project so that errors are limited and debugging is quick.

Validation And Browsers.

By conforming to the standards layed out by W3 and various organizations around the globe, you are ensuring that the browser renders the webpage with the utmost efficiency, and allowing the user to view the page as intended (deviation(s) apply depending on the age of the software version).

My Website, & Validation.

After checking W3 website validation checker I have come to the conclusion that, with the exclusion of the index, my page conforms to the standards of W3.
