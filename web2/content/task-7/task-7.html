JavaScript Exploration.

What is JavaScript?

JavaScript, commonly known as JS, is the backbone of the internet. JS is a general purpose language that, like Java, is a write once, run anywhere language. Javascript can be ran on any machine for any purpose. It can be used for front end visual effects, or for some data data analysis.

JavaScript For Apps.

Don't care for either? What about an app? JavaScript has you covered. JavaScript is the leader when it comes to extentions (also known as libs) to accomplish any task. For your new app idea, you can use libs like Vue, Redux, or React to make the app!

Vanilla JavaScript.

Before you go off learning how to make the next Facebook, there are some things you should know - everything is made on top of the standard JS; so if you don't know what the base JS can do, how will you know how the libs work?

What Can Vanilla JavaScript do?

I may have now made the standard JS feel like it's worthless but in reality, it can still do a lot! There's some examples below that show you exactly what the base version can do, and once you learn the basics, learning any lib becomes so much easier!

JavaScript getMonth()

What is getMonth()? getMonth(), like in Java, is a method. Since JS is a OOP, and general language, it must be able to support classes/objects/methods.
